Sobre el sitio

Este sitio fue desarrollado como un proyecto para poner en practica lo aprendido de HTML y CSS. En si, el sitio trata de mostrar listas simuladas de canciones más escuchadas por país, género, a nivel mundial, etc. Al final es un sitio incompleto debido a los restringidos conocimientos de otros elementos para poder terminar el proyecto, como lo es el uso de una base de datos o hasta la implementación de APIs.
